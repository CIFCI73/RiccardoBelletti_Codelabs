# RiccardoBelletti_Codelabs

Este repositorio contiene mis primeros proyectos hechos en Kotlin usando Android Studio y Jetpack Compose. Estoy empezando todavía, así que estos ejercicios son bastante simples, pero me han ayudado mucho a entender cómo funciona Compose y cómo se estructura una app Android moderna. El primer proyecto que hice fue una pequeña aplicación que solo muestra un mensaje de saludo, un “Hello World” básicamente, usando un composable muy sencillo. Allí aprendí lo más básico: cómo funciona setContent, cómo se crea una función con @Composable, cómo usar textos, colores y un poco de Modifier para poner padding y cosas así. Es simple, pero fue la primera vez que vi realmente cómo funciona Compose en vez del sistema antiguo de layouts XML.

Después hice otro ejercicio que era una tarjeta de cumpleaños con una imagen de fondo y dos textos encima. En este proyecto probé cosas nuevas como cargar imágenes con painterResource, usar Box para superponer elementos y combinarlo con Column para que el texto no se solape. También aprendí a usar recursos de strings y tamaños de texto, que parecen detalles tontos pero en realidad ayudan mucho a organizar el proyecto como se debe hacer en Android. Este ejercicio fue un poco más creativo y me gustó porque me hizo sentir que podía crear interfaces más bonitas sin complicación.

El último proyecto es el más grande de los tres y forma parte del “Basics Codelab”. Aquí trabajé con más componentes como LazyColumn, tarjetas (Card), animaciones con animateContentSize, estados con rememberSaveable y previews en modo oscuro. También tuve que implementar una pantalla de bienvenida que desaparece cuando el usuario pulsa un botón, así que era la primera vez que hacía algo con navegación básica usando solo estado. Me costó un poco al principio porque Compose tiene una manera un poco diferente de pensar la interfaz, pero al final creo que entendí bastante bien el flujo.

Para hacer estos ejercicios me apoyé en apuntes, documentación online y también un poco en ayuda de la IA para entender mejor algunos conceptos y para explicaciones rápidas cuando algo no funcionaba.En general, estos tres proyectos me han servido para empezar bien con Kotlin y Compose, y ahora siento que puedo seguir con ejercicios un poco más avanzados.
